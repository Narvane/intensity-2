# Brand assets

Official Intensity logos for the client app. Place files here; the Vite build imports them from this folder.

| File | Format | Usage |
|------|--------|--------|
| `logo-icon.png` or `logo-icon.svg` | Square | Bootstrap/splash, favicon, native app icon base |
| `logo-wordmark.png` or `logo-wordmark.svg` | Horizontal | Authentication header, horizontal brand contexts |

Prefer SVG when available. If a file is missing, the client shows a design-system placeholder until the asset is added.
